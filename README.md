Walmart Sales Report 

##Over View 

This project is an interactive Power BI dashboard
developed using the Walmart Sales dataset. 
The dashboard provides insights into sales performance,
store-wise analysis, holiday sales, and key economic 
indicators through interactive visualizations.

## Tools Used
- Microsoft Power BI Desktop
- Power Query
- DAX (Data Analysis Expressions)

## Dataset
The dataset contains the following fields:
- Store
- Date
- Weekly Sales
- Fuel Price
- CPI
- Unemployment
- Holiday Flag

## Dashboard Features
- KPI Cards
  - Total Sales
  - Average Sales
  - Store Count
  - Maximum Weekly Sales
- Weekly Sales Trend
- Sales by Store
- Holiday vs Non-Holiday Sales
- Sales Contribution by Store (Treemap)
- Fuel Price vs Weekly Sales (Scatter Chart)
- CPI & Unemployment Trend
- Interactive Slicers (Year, Store, Holiday Status)

## DAX Measures
- Total Sales
- Average Sales
- Store Count
- Maximum Weekly Sales

## Key Insights
- Total Sales: 6.73 Billion
- Average Weekly Sales: 1.05 Million
- Total Stores: 45
- Maximum Weekly Sales: 3.82 Million
- Sales vary across different stores.
- Holiday and non-holiday sales can be compared using the dashboard.
- Fuel price does not show a strong linear relationship with weekly sales.
